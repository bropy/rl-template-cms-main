# App Client

### Client - NextJS

### Host Machine Requirements:
- NodeJS 22.x
- Yarn 1.22.22
- Docker

### Quick commands
#### Quick Start
```sh
make quick-start
```
#### First Start
```sh
make first-start
```

### General
#### Copy environment variables:
```sh
make client-copy-env
```
#### Install dependencies:
```sh
make client-install
```
#### Run in development mode:
```sh
make client-dev
```
#### Code formatting:
```sh
make client-code-check
```

### **[Go to client](http://localhost:3000)**
